# Fix　乐子工具箱 开源学习小程序

基于uni-app、colorUi，支持小程序、H5、Android和IOS,搭建基础，拿来可用的小程序，方便各位开发小主学习借鉴！！！

### 项目说明

1.  项目灵感来源于 前端铺子 ，前台基于Vue-uniApp，使用colorUi与uview框架，部分Demo参考uniApp插件市场等，后台基于“若依”框架,感谢各位作者～

#### 安装教程

1.  下载安装：「HBuildX」、「微信开发者工具」
2.  扫码登录微信开发者工具
3.  将项目拖进【HBuildX】- 运行 - 微信小程序 - 完成
